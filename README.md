# E-Commerce Store

A modern, full-featured e-commerce application built with **React** and **Vite**, demonstrating professional web development practices with a focus on performance, user experience, and clean code architecture.

## 🎯 Key Features

- **Responsive UI** – Built with Tailwind CSS for seamless experience across all devices
- **Product Catalog** – Browse 6+ product categories with advanced filtering capabilities
- **Shopping Cart** – Persistent cart with localStorage, add/remove/quantity management
- **Product Details** – Detailed product pages with comprehensive information
- **User Authentication** – Sign Up & Sign In pages with form validation using React Hook Form
- **Dynamic Routing** – 7+ routes using React Router v7 for smooth navigation
- **State Management** – Context API for efficient global state handling
- **Modern Tech Stack** – Latest React, Vite for optimal build performance, ESLint configuration

## 🛠️ Tech Stack

| Technology          | Purpose                      |
| ------------------- | ---------------------------- |
| **React 19**        | UI Framework                 |
| **Vite**            | Build tool & dev server      |
| **React Router v7** | Client-side routing          |
| **Tailwind CSS**    | Responsive styling           |
| **React Hook Form** | Form management & validation |
| **React Icons**     | Icon library                 |
| **Context API**     | State management             |

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev


## 📁 Project Structure

```

src/
├── assets/ # Images and product data organized by category
├── Components/ # Reusable UI components (Navbar, Cart, ProductCard, etc.)
├── Context/ # StoreContext for global state (products, cart)
├── Pages/ # Page components (Home, Products, ProductDetail, Auth pages)

